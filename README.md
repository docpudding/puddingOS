A high-level configuration layer for NixOS and Home Manager.

## Usage

This project is documented on the repository's [GitHub Pages site](https://docpudding.github.io/puddingOS). There you can find:

- An [Introduction](https://docpudding.github.io/puddingOS/overview/introduction) explaining the purpose of this project.
- An [Installation Guide](https://docpudding.github.io/puddingOS/overview/installation) for various types of systems.
- A [Submodule List](https://docpudding.github.io/puddingOS/configuration/nixos/core) documenting how to configure puddingOS.
